# turbine-core

Turbine Go core modules

## Turbine-rb

From `meroxa/turbine-core/lib/ruby/turbine_rb`.

### Set up

```bash
bundle install
```

### How to run tests

```bash
bundle exec guard
```

### Release
+ Submit a PR incrementing the version in `version.rb`
+ Merge PR after approval
+ Gem is automatically released after merge
